## Commandes utiles

Éteindre (utilisateur `root` ou `sudo`).

```shell
$ shutdown -h now
```

ou

```shell
$ halt
```

Redémarrer (utilisateur `root` ou `sudo`).

```shell
$ shutdown -r now
```

ou

```shell
$ reboot
```
