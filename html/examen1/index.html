1. Stocker les prix dans une structure adaptée (de type méta données), pour pouvoir faire des calculs simplement et rapidement.

2. Écrire une fonction qui recopie les prix stockés depuis la structure dans les cellules du tableau html en les formattant au format français. Vous aurez besoin des fonctions de conversion Number et toLocaleString.

3. Attacher des événements click sur les éléments html input de type checkbox du premier fomulaire pour effectuer le calcul des options.

4. Stocker les prix des options dans une structure adaptée (de type méta données), pour pouvoir faire des calculs simplement et rapidement.

5. Fonctionnement de la méthode de Calcul

  • Recopier les prix dans une structure temporaire de calcul pour ne pas perdre le prix de départ des véhicules
  • Pour chaque case cochée (checked)
    • Si l'option n'est pas disponible, désactiver le calcul du prix final et effacer la cellule
    • Sinon ajouter le prix de l'option au prix du véhicule
  • Recopier les prix calculés dans le tableau html

Moins de 20 lignes de codes sont suffisantes !
