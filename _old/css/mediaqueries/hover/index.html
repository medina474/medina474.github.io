Fonctionnalités d'interaction du média

Atelier Neotech > Feuilles de style > Media Queries > Fonctionnalités d'interaction du média

hover

La capacité du mécanisme de saisie principal à survoler les éléments.
Ajoutée avec le niveau 4 du module de spécification Media Queries.

none
aucun : Le mécanisme d'entrée principal du média n'inclut pas de périphérique de survol.
hover
Indique que le périphérique de pointage principal peut facilement survoler certaines parties de la page. Les exemples incluent les souris et les appareils qui pointent physiquement vers l’écran, comme la manette Nintendo Wii.
```css @media (hover: hover) { } ```
Votre navigateur ne prend pas en charge le query hover
Ce média supporte le survol
Ce média ne supporte pas le survol ```css @media (hover: none) { } ```
https://www.quirksmode.org/css/tests/mediaqueries/hover.html

any-hover

La présence d'un appareil de poinage parmi les mécanismes de saisie et sa précision. Ajoutée avec le niveau 4 du module de spécification Media Queries.

Votre navigateur ne prend pas en charge le query any-hover
Ce média possède au moins un périphérique supportant le survol
Ce média ne possède aucun périphérique supportant le survol

pointer

none
aucun : Le mécanisme d'entrée principal du média n'inclut pas de périphérique de pointage.
coarse
grossier : Le mécanisme d'entrée principal du périphérique comprend un périphérique de pointage d'une précision limitée. Les exemples incluent les écrans tactiles et les capteurs de détection de mouvement (Télécommande Wii, Kinect, ...).
fine
fin : Le mécanisme d'entrée principal du périphérique comprend un périphérique de pointage précis. Les exemples incluent les souris, les pavés tactiles (trackpad) et les tablettes tactiles avec stylets de dessin.
Votre navigateur ne prend pas en charge le query pointer
Ce média supporte le pointage fin
Ce média supporte le pointage grossier
Ce média ne supporte pas le pointage
https://www.quirksmode.org/css/tests/mediaqueries/pointer.html

any-pointer

La capacité d'un des mécanismes de saisie à survoler les éléments
Ajoutée avec le niveau 4 du module de spécification Media Queries.

Votre navigateur ne prend pas en charge le query any-pointer
Ce média possède au moins un périphérique supportant le pointage fin
Ce média possède au moins un périphérique supportant le pointage grossier
Ce média ne possède aucun périphérique supportant le pointage
